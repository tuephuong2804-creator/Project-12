# Sentiment Analysis Classification Project

## Description
This project performs sentiment classification using a Twitter dataset.

## Methods
- Data preprocessing and cleaning
- TF-IDF vectorization with bigram
- Logistic Regression model

## Results
The model predicts three classes: Positive, Neutral, and Negative.  
Results are visualized using a bar chart.

## How to run
pip install -r requirements.txt  
python main.py
